# Lecture 1: What is an API?

**Definition:** 
An API (Application Programming Interface) is a set of rules and protocols that allows different software applications to communicate with each other.

**Simple words mein (Hinglish):**
API ek middleman (ya waiter) ki tarah kaam karta hai. 
Socho tum ek restaurant mein ho. Tumhe kitchen mein ja kar apna khana khud nahi banana hai, tum waiter ko apna order batate ho. Waiter tumhara order kitchen mein le jata hai, aur waha se tayyar khana tum tak wapas lata hai.

Yaha par:
- **Tum (Client):** Jo request kar raha hai (jaise tumhara mobile app ya website).
- **Kitchen (Server/Database):** Jaha par saara data aur logic hota hai.
- **Waiter (API):** Jo tumhari request kitchen tak le jata hai, aur kitchen ka response tumhe wapas la kar deta hai.

Toh API bas ek zariya hai jisse do alag-alag softwares aapas mein baat kar sakte hain. Jaise jab tum Ola/Uber app kholte ho toh wo Google Maps ki API use karke tumhe map dikhata hai.
