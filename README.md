# FastAPI Zero to Hero 🚀

Welcome to the **FastAPI Learning Repository**! 

Is repository ka main purpose hai FastAPI ko bilkul basic se aur aasaan (simple) language mein samajhna. Agar aap backend development mein naye hain ya kisi aur framework se FastAPI par switch kar rahe hain, toh ye repository aapke liye ek perfect guide banegi.

## 🗺️ FastAPI Learning Roadmap

Neeche ek step-by-step roadmap diya gaya hai jisko follow karke hum is repository mein aage ke notes aur code add karenge:

### Phase 1: Basics & Setup (Jahan se humne shuru kiya)
- [x] **API kya hai?** (Waiter & Kitchen example)
- [x] **FastAPI kya hai?** (Core philosophy)
- [x] **Synchronous vs Asynchronous ka matlab** (Basic examples)
- [x] **Virtual Environment (Macbook) setup karna**
- [x] **FastAPI ka correct import syntax aur Pehli API likhna**

### Phase 2: Core Concepts (Next hum kya cover karenge)
- [ ] **Path Parameters & Query Parameters:** URL se data kaise bhejein.
- [ ] **Pydantic Models (Request Body):** Users se form/JSON data kaise receive karein.
- [ ] **Data Validation:** Galat data aane par API khud usko kaise reject kare.

### Phase 3: Database & Structure
- [ ] **SQLAlchemy Setup:** Database (SQL) ke sath connect karna.
- [ ] **CRUD Operations:** Create, Read, Update, Delete queries banana.
- [ ] **Routers (APIRouter):** Bade project ko alag-alag files mein todna (Structuring).

### Phase 4: Advanced & Security
- [ ] **Middleware & CORS:** Kaun hamari API access kar sakta hai (Security).
- [ ] **Authentication (JWT & OAuth2):** Login aur Signup system banana (Tokens).
- [ ] **Background Tasks:** Time-consuming kaam ko background mein chalana.

### Phase 5: Deployment
- [ ] **Dockerize FastAPI:** Project ko container mein daalna.
- [ ] **Deploy to Cloud:** Deta/Render/AWS par apni API ko live karna.
